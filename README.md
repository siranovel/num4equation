## install ##

From rubygems:  
~~~
    [sudo] gem install num4equ
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/num4equation.git  
    cd num4equation 
    jruby -S gem build num4equ.gemspec
    jruby -S gem install num4equ
~~~
